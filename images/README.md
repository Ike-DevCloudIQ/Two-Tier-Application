# Screenshots Directory

This directory contains deployment evidence screenshots for the Two-Tier AWS Infrastructure project.

## Required Screenshot Files:
- `terraform-plan.png` - Terraform planning phase output
- `vpc-resource-map.png` - AWS VPC Console resource map
- `nat-gateways.png` - NAT Gateway deployment
- `launch-templates.png` - EC2 Launch Templates
- `auto-scaling-groups.png` - Auto Scaling Groups configuration
- `load-balancer.png` - Application Load Balancer
- `rds-aurora.png` - RDS Aurora cluster
- `route53.png` - Route 53 DNS configuration

## How to Add Screenshots:
1. Save your AWS console screenshots to this directory with the names above
2. The README.md will automatically reference these images
3. Commit and push the changes to GitHub
